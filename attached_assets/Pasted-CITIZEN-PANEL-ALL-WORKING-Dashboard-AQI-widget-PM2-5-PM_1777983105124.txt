CITIZEN PANEL — ALL WORKING
Dashboard: AQI widget (PM2.5/PM10/NO2/CO colored badges), Ward Health Score (0-100 circular ring, calculated: 40% resolution rate + 30% AQI + 30% response time), quick actions grid (SOS/Complaint/Map/Bills/Schemes/Community), last 3 complaints with status.
Complaint Filing (4 steps):

Text area → after 3 words call OpenAI → returns category + priority 0-100 + title suggestion → show AI badge in real-time
Leaflet map pin + Nominatim reverse geocode → auto-fill address
Photo upload → call OpenAI Vision → returns detected issue + severity → auto-fill fields
Review + submit. After submit: animated success, redirect to tracker.
Complaint tracker: timeline (Pending→Assigned→In Progress→Resolved) with timestamps. Chat button when assigned.

SOS: Full-screen red page. Large pulsing button. 3s countdown with cancel. On activate: navigator.geolocation watchPosition, emit socket 'sos:trigger', send GPS every 15s via 'sos:location_update'. Shake detection (DeviceMotionEvent >25m/s²). Show "SOS ACTIVE — Help is coming" with live coordinates.
Live Map (Leaflet): Toggle layers: police stations (blue pins, popup with call button), active SOS (red pulsing), risk zones (colored polygons — red=crime/orange=flood/yellow=garbage), complaint pins by category, user location blue dot.
Schemes: Cards with filter by category. Detail page: eligibility checklist, benefits, documents list with checkboxes, step-by-step apply guide, "Check Eligibility" → AI chat pre-loaded with scheme context.
Bills: Cards for Electricity/Water/MCD/Traffic. Payment flow: mock UPI/card, PIN 1234, 2s processing animation, receipt with reference number saved to DB.
Voting & Polls: Active polls for user's ward. Vote → animated bar chart reveals. Past polls with results.
Petitions: List with signature progress bars. Sign button. Create petition form. Detail page shows escalated status when threshold met + official response when available.
Voice Complaint: Mic button on complaint page → MediaRecorder API → send to OpenAI Whisper → transcription auto-fills description → AI classifies. Fallback: Web Speech API.
Community Forum: Ward-filtered posts. Create post (category/title/body/photo). Like + comment. Tabs: All/Issues/Events/Help/General.
Civic Event Calendar: CSS grid monthly calendar, colored dots on dates, click → events panel. RSVP toggle with live count via socket. Reminder via browser notification.
RTI Filing: 4-step wizard. Step 1: select department + topic. Step 2: "Draft with AI" → OpenAI generates legal RTI text ("Under RTI Act 2005, Section 6...") in editable textarea. Step 3: PIO details (seeded by dept). Step 4: submit → deadline = today+30 days. Status tracker. Cron: alert 3 days before deadline.
Emergency Services: Tabs (Hospital/Ambulance/Police/Fire/Blood Bank/Shelter). Sorted by distance (Haversine from GPS). 1-tap call (tel: link). Map view toggle.
Ward Report Card: Monthly metrics: resolution rate % vs city avg, avg resolution time, AQI trend line chart, top issues pie chart, officer response rate. Grade A-F. Previous months dropdown.
Citizen-Officer Chat: WhatsApp-style bubbles. Real-time via Socket.io room complaint_${id}. Read receipts. Typing indicator. Photo sharing. Opens from complaint detail when status ≠ pending.
AI Chatbot: Floating button all pages. Slide-up panel. OpenAI with civic system prompt. Suggested prompts on first open. Streaming response. Persist history in localStorage per session.
Gamification: Points: file complaint +10, resolved +20, sign petition +5, create escalated petition +50, RSVP+attend event +15, file RTI +25. Leaderboard: city-wide top 10 + ward top 10 + user's rank highlighted. Badges on profile.

ADMIN PANEL — ALL WORKING
War Room Dashboard: Live KPIs (Socket.io): total complaints today/week/month, active SOS count (red pulse if >0), resolution rate %, avg resolution time, active workers. AI Insights panel (every 5min, OpenAI analyzes complaint data → 3-5 insight strings). Live activity feed (right sidebar, socket stream). Recharts: donut chart by category, bar chart ward comparison, line chart 30-day trend.
SOS Management: Active alerts sorted oldest-first. Each card: user name/phone/time elapsed/last location. Incident detail: live map with location polyline trail (socket updates), user info, nearest police stations with dispatch button (creates notification to police), timeline, Mark Responding / Mark Resolved actions. History tab with response time metrics.
Complaint Management: Table with columns: category icon, title, ward, priority bar, status badge, worker, SLA status (on-track/at-risk/breached). Filters: status/category/priority range/ward/date range. Bulk assign/resolve/reject. Full-text search. Pagination 20/page. Detail slide-over: full info, photo, AI reasoning, assign worker dropdown (available + skilled + nearest), SLA countdown. Cluster map view (Leaflet marker clustering).
Field Worker Management: Live map (Leaflet): worker pins (green=available/yellow=on_task/gray=off_duty), auto-refresh 30s. List table with assign/call. Worker detail: performance chart, task history, ratings. Workers ping location every 30s via POST /api/workers/ping.
Emergency Broadcast: Title + message composer. Target: all wards or specific ward(s). Type badge (Info/Warning/Emergency/Event). Preview card. Send → socket 'broadcast:new' → toast to all targeted users. History with read count.
AI Auto-Assignment (default ON, toggle in settings): On new complaint: find workers where status=available AND wardId matches AND skills include category → sort by Haversine distance from complaint → assign nearest. If none available: flag "Needs Manual Assignment" + admin notification.
Predictive Maintenance: Daily 6am cron → send 30-day complaint frequency data to OpenAI → returns predictions array (ward, issue, confidence%, timeframe, recommended action). Show as prediction cards. "Create Preventive Task" button. Accuracy tracking.
SLA Auto-Escalation (cron every 30min):

Pothole High: 24h, Pothole Normal: 72h, Garbage: 48h, Water: 12h, Electricity: 6h, Streetlight: 48h
50% elapsed → warn worker (level 1)
100% (breached) → alert ward officer (level 2)
+24h → alert dept head (level 3)
+48h → alert commissioner + public flag on report card (level 4)

Officer Accountability: Per-officer cards: assigned vs resolved, avg time, citizen rating, badge (Outstanding/Good/Needs Improvement/Critical). Detail page: monthly performance Recharts line, escalations caused. Monthly auto-report to officer + supervisor (cron 1st of month).
Heatmap Analytics: Full-screen Leaflet map. Layer toggles: complaint density choropleth, SOS clusters, AQI gradient, resolution rate by ward, category heatmaps. Sidebar charts: ward comparison, time-of-day 24×7 grid heatmap, top 10 worst wards. CSV export.
Budget & Project Tracker: Project cards: name/ward/budget bar (allocated vs spent)/timeline/completion %/status. Linked complaint IDs. Add project form. Financial overview: Recharts donut + bar charts.
